`substrateminer` requires the following dependencies:
- Python >= 3.10.11
- BioPython >= 1.84
- Numpy
- SciPy
- Pandas
- matplotlib <= 3.6.0
- weblogo
- requests
- click
- PyYAML
- pillow
Optional binary dependencies for multiple sequence alignment:
- Clustal Omega >= 1.2.4
- MUSCLE >= 5.1
- MAFFT >= 7.475
